## Lancement du Projet

Pour démarrer le projet, suivez ces étapes pour installer et lancer le projet :

### Prérequis

- XAMPP installé sur votre machine

### Installation et Exécution

**Initialiser package.json** :

   ```bash
   npm init -y
   ```

**Installer les dépendances** :

   ```bash
   npm install
   ```

**Lancer le serveur** :

   ```bash
   npm run dev
   ```
